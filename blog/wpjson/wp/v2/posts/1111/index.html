To aid in an answer on StackOverflow<\/a> that I had recently answered, I\u2019m providing part of the response here.<\/p>\n

The question was essentially, \u201cwhat\u2019s a way to use DataTriggers in Silverlight, without DataTriggers?\u201d<\/p>\n

I had suggested one idea would just to use VisualStates and a code behind file.<\/p>\n

That\u2019s what I\u2019ve done here. I created an enum of type AnimateState, which has three possible values, Top, Left, and Right. By clicking on one of three buttons on the simple UI, it changes the value of the property, which in turn calls one of the VisualStates defined in the XAML.<\/p>\n

In the example, it animates the position of the orange ellipse to various positions on the canvas. <\/p>\n
