I recently upgraded my old Linksys router to the custom firmware, Tomato<\/a> (definitely recommend!). (I had been running DD-WRT<\/a>, which is powerful, but has a much more complex user interface). After upgrading the router, I was unable to connect to my Slingbox. (As an aside, I\u2019m not 100% confident that there\u2019s a cause\/effect there \u2013 but it\u2019s one key thing that changed recently). <\/p>\n

The error was:<\/p>\n

Error: 0x9236000d
Context: 20
Operation: 110<\/p>\n

The online help suggested that something might be wrong and I should fix it before continuing. Duh<\/strong>. <\/p>\n

After experimenting with some firewall settings, IP address changes, and a few more pointless attempts, I found that the only thing that fixed the connectivity issues was to completely reset<\/a> my Slingbox and run through the setup again. The Sling Player software immediately recognized the \u201cunconfigured\u201d Sllngbox as soon as the reset finished. I reapplied all of the settings (including resetting the passwords back to their old values), and I\u2019m again able to connect and use the Slingbox with no issues.<\/p>\n
