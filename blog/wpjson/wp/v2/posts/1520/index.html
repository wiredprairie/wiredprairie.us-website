<\/a>More than a few years ago, I created SnugUp<\/a> version 1, which is a handy way of synchronizing folders of images with SmugMug<\/a> for Windows users. I\u2019ve made a number of changes in the last month based on some requests and the result is a significant update (yet the core features are all there \u2013 just improved!).<\/p>\n

New features:<\/p>\n

    \n
  • (Changed to version 2)<\/li>\n
  • Completely revamped look and feel (which did remove a few ‘flashy’ features) <\/li>\n
  • Uses Click once for application updating (which should make it easier to push updates and bug fixes out to users)<\/li>\n
  • Added settings for automatic upload <\/strong>(when application starts) <\/li>\n
  • Added setting for Subcategory selection (only selection, no creation of subcategories through SnugUp) <\/li>\n
  • Added setting for custom gallery naming <\/strong><\/li>\n
  • Completely changed settings user interface to more logically group and explain settings <\/li>\n
  • Fixed a number of bugs (that shouldn’t have appeared to user anyway!) <\/li>\n
  • Added support for new file extensions (to upload videos <\/strong>in particular)<\/li>\n<\/ul>\n

    It does require .NET 4.0 Framework Client profile (which likely you\u2019ve already got on your machine, but if you don\u2019t the installer SHOULD make it easy to download).<\/p>\n

    Go here<\/a> (http:\/\/www.wiredprairie.us\/SnugUp<\/a>) for more information and to download. I\u2019d recommend uninstalling the old version first.<\/p>\n
