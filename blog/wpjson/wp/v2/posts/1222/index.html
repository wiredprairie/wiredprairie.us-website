Within the past few days–maybe after a recent Windows Update–one of my laptops started to fail to successfully connect to the WHS 2011 server I\u2019ve got in my home. The error was \u201cthe server appears to be offline\u201d and asked if I\u2019d instead want to connect off-line. (Yeah, no thanks.)<\/p>\n

A bit of hunting on the Internet later \u2013 no one had a clear path to success. Most reported continual failures when trying to start the LaunchPad or start a backup manually. <\/p>\n

I had success by doing the following:<\/p>\n

    \n
  1. Start<\/strong> > View Local Services<\/strong> (you can type \u201cservices\u201d for a quick match)<\/li>\n
  2. Find Windows Server Health Service<\/strong> and start it if it isn\u2019t already started. <\/li>\n
  3. Find Windows Server Client Computer Backup Provider Service<\/strong> and start it if it isn\u2019t already started.<\/li>\n<\/ol>\n

    <\/p>\n

    <\/p>\n

    <\/p>\n

    Neither of these were running on my laptop even though they both were set to Automatic startup. As soon as I started them manually, the LaunchPad immediately began working, and manual backup became available again.<\/p>\n

    There were no errors in the Windows Event Viewer explaining why they may have failed to start. As I don\u2019t reboot my laptop frequently, this fix will remain in place for a while (and I\u2019ll need to remember to check these in the future potentially). <\/p>\n
