The developer sessions were generally very good. Some were absolutely better than others, a few really needed my tips for doing a better presentation<\/a>. A new one — just because you can use zoom — don’t use it as a crutch. If you’re planning some demos, create them using big fonts to begin with. I saw a decent presentation today about advanced Reflection, yet it was marred by a demo tool that was designed to be read at a distance of 2 foot. The audience had to trust the presenter that what he said was true. It was unfortunately.<\/p>\n
