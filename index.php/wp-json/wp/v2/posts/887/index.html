\n

The crack relies on a weakness so astoundingly bone-headed that it\u2019s almost hard to believe. While the data on the drive is indeed encrypted using 256-bit crypto, there\u2019s a huge failure in the authentication program. When the correct password is supplied by the user, the authentication program always send the same character string to the drive to decrypt the data no matter what the password used. What\u2019s also staggering is that this character string is the same for Kingston, SanDisk and Verbatim USB flash drives.<\/p>\n<\/blockquote>\n
