The software is updated frequently, they respond on twitter<\/a> quickly, \u2026 a great little company based in Canada.<\/p>\n

I\u2019ve hooked the application\u2019s password store to my DropBox<\/a> account for easy multiple-machine and OS synchronization. <\/p>\n

Notes is sort of the \u201ccatch all.\u201d <\/p>\n

The only thing that bothers me is how nice the OSX version looks and how antiquated the Windows version is. On Twitter, I referred to it as \u201cbeing hit by an ugly stick.\u201d<\/p>\n

It\u2019s a shame too \u2013 Windows applications don\u2019t need to look like a bad version of the Windows File Explorer. It baffles me as to why it doesn\u2019t look more like the Mac version. From a quick exploration of the way the EXE was built \u2013 (hey, remember where you\u2019re reading this) it looks like they may have used Delphi! I know Delphi can do better than this \u2013 I was writing Delphi 1 applications that used background texturing, 256 bit color (yes, yes, that was actually a big deal back in the day), fancy rounded buttons\u2026 (yeah yeah, I know).<\/p>\n
